Проект открывать с помощью среды разработки IntelliJ IDEA.

Как открыть?
1.Разархивировать файл, в любое место (главное запомните это место)
1.Заходим в IntelliJ IDEA
2.Нажимаем кнопку File
3.Дальше Open
4.Потом идёт туда где хранится ваш файл, например, если он находится в загрузках: C:\Users\Сергей\Downloads\LB4
5.Теперь когда мы дошли до папки где хранятся файлы проекта, нажимаем кнопку OK
